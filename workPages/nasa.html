Work Overview

NASA Kennedy NE-X

My work in the NE-X branch at the Kennedy Space Center concerned ISRU (In Situ Resource Utilization). My work with machine learning was on SWAMPWORKS RASSOR (Regolith Advanced Surface Systems Operations Robot). The simulations I built with Unity concerned general ISRU activity and existed for educational purposes. Below are images of RASSOR and my white paper on a machine learning script I wrote for mass prediction on RASSOR.

NASA Kennedy UB-T

While in the UB-T branch, I built tools to assist in lunar mission planning. These tools consisted mainly of a Unity simulation/data request tool as well as multiple python data analysis scripts.


Developed a tool with Unity and C# for lunar lander mission planning.

Built a query system for lunar temperature and lighting data based on different measurements of time and position.

Performed data analysis and built models with Python using LOLA and DIVINER temperature, lighting, and elevation data.
